Transition

  • transition-property:
    1. none Отсутствие свойства для перехода.
    2. all Значение по умолчанию. Применяет эффект перехода ко всем свойствам элемента.
    3. свойство Определяет список CSS-свойств, перечисленных через запятую, участвующих в переходе.
    4. initial Устанавливает значение свойства в значение по умолчанию.
    5. inherit Наследует значение свойства от родительского элемента.
  • transition-duration:
    1. время Время перехода указывается в секундах или миллисекундах, например, 2s или 5ms.
    2. initial Устанавливает значение свойства в значение по умолчанию.
    3. inherit Наследует значение свойства от родительского элемента.
  • transition-timing-function:
    1. ease Функция по умолчанию, переход начинается медленно, разгоняется быстро и замедляется в конце. Соответствует cubic-bezier(0.25,0.1,0.25,1).
    2. ease Функция по умолчанию, переход начинается медленно, разгоняется быстро и замедляется в конце. Соответствует cubic-bezier(0.25,0.1,0.25,1).
    3. ease-in Переход начинается медленно, а затем плавно ускоряется в конце. Соответствует cubic-bezier(0.42,0,1,1).
    4. ease-out Переход начинается быстро и плавно замедляется в конце. Соответствует cubic-bezier(0,0,0.58,1).
    5. ease-in-out Переход медленно начинается и медленно заканчивается. Соответствует cubic-bezier(0.42,0,0.58,1).
    6. cubic-bezier(x1, y1, x2, y2) Позволяет вручную установить значения от 0 до 1 для кривой ускорения. На этом сайте вы сможете построить любую траекторию перехода.
    7. initial Устанавливает значение свойства в значение по умолчанию.
    8. inherit Наследует значение свойства от родительского элемента.
  • transition-delay:
    1. время Время задержки перехода указывается в секундах или миллисекундах.
    2. initial Устанавливает значение свойства в значение по умолчанию.
    3. inherit Наследует значение свойства от родительского элемента.

Transform

  • Функции 2D-трансформации transform:
    1. none Значение по умолчанию, означает отсутствие трансформации.
    2. matrix(a, c, b, d, x, y) Смещает элементы и задает способ их трансформации, позволяя объединить несколько функций 2D-трансформаций в одной.
    3. translate(x,y) Сдвигает элемент на новое место, перемещая относительно обычного положения вправо и вниз, используя координаты X и Y, не затрагивая при этом соседние элементы.
    4. translateX(n) Сдвигает элемент относительно его обычного положения по оси X.
    5. translateY(n) Сдвигает элемент относительно его обычного положения по оси Y.
    6. scale(x,y) Масштабирует элементы, делая их больше или меньше. Значения от 0 до 1 уменьшают элемент.
    7. scaleX(n) Функция масштабирует элемент по ширине, делая его шире или уже.
    8. scaleY(n) Функция масштабирует элемент по высоте, делая его выше или ниже.
    9. rotate(угол) Поворачивает элементы на заданное количество градусов, отрицательные значения от -1deg до -360deg поворачивают элемент против часовой стрелки, положительные — по часовой стрелке.
    10. skew(x-угол,y-угол) Используется для деформирования (искажения) сторон элемента относительно координатных осей.
    11. skewX(угол) Деформирует стороны элемента относительно оси X.
    12. skewY(угол) Деформирует стороны элемента относительно оси Y.
    13. initial Устанавливает значение свойства в значение по умолчанию.
    14. inherit Наследует значение свойства от родительского элемента.
  • Точка трансформации transform-origin:
    1. ось Х(left, center, right, длина, %) ось Y(top, center, bottom, длина, %) Пара значений, заданная с помощью ключевых слов, единиц длины или процентов определяет, относительно какой части элемента будет происходить трансформация.
    2. initial Устанавливает значение свойства в значение по умолчанию.
    3. inherit Наследует значение свойства от родительского элемента.
  • Множественные трансформации:
    1. Можно объединить несколько трансформаций одного элемента, перечислив их через пробел в порядке проявления.

Animation

  • animation-name:
    1. none Означает отсутствие анимации.
    2. имя анимации Имя анимации, которое связывает правило @keyframes с селектором.
    3. initial Устанавливает значение свойства в значение по умолчанию.
    4. inherit Наследует значение свойства от родительского элемента.
  • animation-duration:
    1. время Указывает время, которое анимация занимает для завершения одного цикла.
  • animation-timing-function:
    1. Свойство animation-timing-function описывает, как будет развиваться анимация между каждой парой ключевых кадров. Во время задержки анимации временные функции не применяются.
  • animation-iteration-count:
    1. infinite Анимация проигрывается бесконечно.
    2. число Анимация будет повторяться указанное количество раз. Если число не является целым числом, анимация закончится в середине последнего цикла.
  • animation-direction:
    1. normal Все повторы анимации воспроизводятся так, как указано. Значение по умолчанию.
    2. reverse Все повторы анимации воспроизводятся в обратном направлении от того, как они были определены.
    3. alternate Каждый нечетный повтор цикла анимации воспроизводятся в нормальном направлении, каждый четный повтор воспроизводится в обратном направлении.
    4. alternate-reverse Каждый нечетный повтор цикла анимации воспроизводятся в обратном направлении, каждый четный повтор воспроизводится в нормальном направлении.
    5. initial Устанавливает значение свойства в значение по умолчанию.
    6. inherit Наследует значение свойства от родительского элемента.
  • animation-play-state:
    1. running Анимация выполняется. Значение по умолчанию.
    2. paused Анимация приостанавливается. При перезапуске анимация начинается с того места, где она была остановлена, как если бы «часы», управляющие анимацией, остановились и снова запустились.
    3. initial Устанавливает значение свойства в значение по умолчанию.
    4. inherit Наследует значение свойства от родительского элемента.
  • animation-delay:
    1. время Время определяет длительность задержки между началом анимации (когда анимация применяется к элементу через свойства) и когда она начинает выполняться.
  • animation-fill-mode:
    1. none Значение по умолчанию.
    2. forwards После того, как анимация заканчивается (как определено значением animation-iteration-count), анимация будет применять значения свойств к моменту окончания анимации.
    3. backwards В течение периода, определенного с помощью animation-delay, анимация будет применять значения свойств, определенные в ключевом кадре, которые начнут первую итерацию анимации.
    4. both Позволяет оставлять элемент в первом ключевом кадре до начала анимации (игнорируя положительное значение задержки) и задерживать на последнем кадре до конца последней анимации.
